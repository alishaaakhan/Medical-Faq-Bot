# Medical-Faq-Bot
🩺 MediCare AI – Medical FAQ Assistant
MediVerse AI is an intelligent Medical FAQ Assistant built using Streamlit, Gemini AI, FAISS, and RAG (Retrieval-Augmented Generation).
It allows users to upload healthcare-related PDF documents and ask medical questions based strictly on the uploaded content.
The system uses semantic search and AI-powered retrieval to generate accurate, context-aware, and safe healthcare responses.
🚀 Features
📄 Upload Medical PDF Documents
🧠 AI-Powered Semantic Search
⚡ Fast FAISS Vector Retrieval
🤖 Gemini AI Response Generation
🏥 Medical FAQ Assistant
📚 Context-Based Answers
🔍 Source Context Viewer
🛡️ Safe Medical Response System
✨ Premium Modern Healthcare UI
🎨 Responsive and Attractive Frontend
